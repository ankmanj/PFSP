# graph_tool
AMSP graph tool based on node_editor for visualization of AMSP application. 

# Compiling and Running
1. Please make sure simple_run.sh has executable rights
    ```
    chmod +x ./simple_run.sh
    ```
2. Run the shell script 
   ```
   ./simple_run.sh
   ```
